Basic Layout:

To seperate different elements, we use containers. Container just means somewhere where all other elements are stored.

in HTML we use "<div>" tags, ofcourse dont forget to close the tag with a second tag and a "/" at the start!

Every Container and/or Tag have to be within the "<body>" Tag, they also go to the next line if you have multiple divs

(well technically they dont HAVE to, but if you want to have clean and working code in the future it does)

If you're wondering why you need containers, they will become very handy once you start with css aka the optic part of websites, also as said they automatically go to the next line like "<p>"
