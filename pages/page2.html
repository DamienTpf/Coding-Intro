<p>

You use this tag to write paragraphs, after the "<p>" tag closes, a new line automatically gets entered, just write the contents inside.

<h1> - <h6>

Simple header tags, so you dont need to do the whole css for it. the bigger the number in the tag the smaller the text, only 6 headers pre-configured

<a>

Used to Link to outside sources, most often written as <a href="https://some-link.de">, also works with local files. automatically recolors everything inside the tag to a dark blue (I changed the color to green), as to show its a link.
The text which contains the Link should be written between the start & end tag
<a href="https://twitter.com/">Twitter</a>
Twitter

<br>

Simple break, people dont like to see this since its "unclean code", so try to start a new line with actual tags not breaks

<img>

Usually written as <img src="https://some-link.de">, as the tag-name tells, you just insert an image.
Doesn't require a closing tag.

<table>

Tables are a bit more complicated, "<table>" is just the "main" tag, you will also need <tr> (Table Row) <thead> (Table Head) <tbody> (Table Body) & <td> (Table Data)

<form>

Forms are there, so you can actually input stuff into your website. like Login-Data or maybe like Feedback! And while you can create Input Fields without the <form>-Tag, you will need it to actually work with the data correctly. (with a few exceptions, but we talk generally here!)

Explanation

A <form> Tag includes many other types of tags, but most prominently the <input>-tag , it's used to access all different kinds of inputs!

A standard <input>-tag will always be an Text-Field input field like:

but by setting the Attribute (the thing you set outside the tag name but inside the Tag), you can alter the Type of input field, that you want to use.

All Input Types

These input types can reach from a color-Picker to a File, so almost everything should be covered here!

if you want to actually use the data you will need a submit button though, so try to include a <input type="submit"> in your <form>!

Sadly there are more types of inputs you will have to know if you work with forms, <Textarea> is a simple one, as the name says you have a block of text to work with, that you can also customize more!

To get a Selection going use <Select> with <option> tags inside of them for the different options.
and with the "selected" attribute (<option selected>), you can automatically assign an option, even if it isn't the first one!

While not an Input field, you can wrap a <label> Tag around any input field to give it as you may guess, a label.

Like this: <label>Name<input type="text"></label>
